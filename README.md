Description:
A fun and interactive Simon Game built using HTML, CSS, and JavaScript, where players test their memory by repeating an increasingly complex sequence of colors and sounds. The project demonstrates logic building, event handling, DOM manipulation, and game state management in JavaScript.

Features:
Dynamic color sequence generation
User interaction with visual and sound feedback
Score tracking and level progression
Fully responsive and visually engaging UI

Tech Stack:
HTML
CSS
JavaScript

How to Play:
Press any key to start the game.
Watch the color sequence shown.
Repeat the sequence in the correct order.
Each level adds one new color to the pattern.
